static load (   $className)
static

处理自动加载类,通过调用use方法，获得namespace class的路径 在Suara的namespce命名规则中， 以Suara开头，后面紧跟着类型 该类别将会自动解析成相应的路径。目前支持三种: libs、 modules、 models。 之后为class所在的相对路径，比如说： Suara 经过自动能解析后会变成 SUARA_CORE_PATH/libs/Core/Configure.php;

具体关于namespace可以直接参考php官方文件。

参见
http://www.php.net/manual/en/language.namespaces.php
