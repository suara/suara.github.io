执行更新记录操作

参数
$data要更新的数据内容，参数可以为数组也可以为字符串，建议数组。 为数组时数组key为字段值，数组值为数据取值 为字符串时[例：name='phpcms',hits=hits+1]。 为数组时[例: array('name'=>'phpcms','password'=>'123456')] 数组的另一种使用array('name'=>'+=1', 'base'=>'-=1');程序会自动解析为name = name + 1, base = base - 1
